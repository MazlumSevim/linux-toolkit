# 🛠️ Linux Toolkit

Ein kleines Linux-Administrationstool für die Verwaltung und Überwachung eines Linux-Systems.

## 📁 Projektstruktur

```text
linux-toolkit/
├── scripts/
├── logs/
├── docs/
├── backup/
└── README.md
```

## 🎯 Projektziele

- Arbeiten mit Linux-Befehlen
- Bash-Skripte erstellen
- Git zur Versionsverwaltung nutzen
- Änderungen dokumentieren
- Administrative Aufgaben automatisieren

## 📂 Verzeichnisse

### scripts
Enthält alle Bash-Skripte.

### logs
Speichert Log-Dateien der Skripte.

### docs
Enthält Dokumentationen.

### backup
Speichert Sicherungen und Backups.

## 🚀 Verwendete Technologien

- Linux
- Bash
- Git
- GitHub
